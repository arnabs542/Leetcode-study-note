## Problem description
```
Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

Integers in each row are sorted from left to right.
The first integer of each row is greater than the last integer of the previous row.
Example 1:

Input:
matrix = [
  [1,   3,  5,  7],
  [10, 11, 16, 20],
  [23, 30, 34, 50]
]
target = 3
Output: true
Example 2:

Input:
matrix = [
  [1,   3,  5,  7],
  [10, 11, 16, 20],
  [23, 30, 34, 50]
]
target = 13
Output: false
```
## 4/2/2020 Binary search

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        //if the matrix is empty
        if(matrix == null || matrix.length == 0 || matrix[0].length == 0) return false;
        int n = matrix.length * matrix[0].length;
        //binary search
        int l = 0, r = n - 1;
        while(l < r){
            int m = l + (r - l) / 2;
            if(num(matrix, m) >= target) r = m;
            else l = m + 1;
        }
        return num(matrix, l) == target;
    }
    //convert absolute index to matrix index
    private int num(int[][] matrix, int index){
        return matrix[index / matrix[0].length][index % matrix[0].length];
    }
}
```
